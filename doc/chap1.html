1 Overview

CurlInterface provides functionality which allows URLs to be downloaded from the internet.

1.1 Functions

At present, curlInterface provides a single function:

1.1-1 DownloadURL
‣ DownloadURL( URL[, verifyCert] )( function )

Returns: A record describing the result. This record will always contain the option 'success', which will be true or false. If 'success' is true, then 'result' will contain the downloaded file. If 'success' is false, then 'error' will contain a human-readable string describing the error.

Download a URL from the internet. Accepts a web address as a string, which can start with either of http:// or https:// The optional argument verifyCert can be set to false to disable verification of HTTPS certificates. If not specified, it defaults to true.
